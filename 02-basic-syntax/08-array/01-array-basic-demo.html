JavaScript 数组基础

数组注意事项

  • 数组定义
    • 使用方括号 [] 或 new Array()
    • 可以包含不同类型元素
    • 长度动态可变
  • 元素访问
    • 使用索引从0开始
    • 越界访问返回undefined
  • 最佳实践
    • 优先使用字面量语法
    • 避免稀疏数组
    • 使用length属性获取数组长度

数组基础示例

// 1. 数组定义
const arr1 = [1, 2, 3]; // 字面量语法
const arr2 = []; // 空数组(构造函数用法已移到专门demo)
const mixedArr = [1, 'two', true, {name: '张三'}]; // 混合类型

// 2. 元素访问
console.log(arr1[0]); // 1 (第一个元素)
console.log(arr1[arr1.length - 1]); // 3 (最后一个元素)
console.log(arr1[5]); // undefined (越界访问)

// 3. 数组长度
console.log(arr1.length); // 3
arr1.length = 5; // 扩展数组
console.log(arr1); // [1, 2, 3, empty × 2]

// 4. 遍历数组
for (let i = 0; i < arr1.length; i++) {
    console.log(arr1[i]);
}
