JavaScript 函数是对象

函数作为对象的特点

在JavaScript中，函数实际上是对象，具有以下特性：

  • 可以拥有属性和方法
  • 可以作为参数传递
  • 可以作为返回值
  • typeof 返回 "function"
  • instanceof Object 返回 true
  • 有内置的 call/apply/bind 方法

基础示例

1. 函数可以拥有属性

function greet() {
    return `Hello, ${greet.name}!`;
}

// 添加属性
greet.language = 'English';
greet.author = 'John Doe';
点击按钮查看结果

2. 函数可以作为参数传递

function process(input, processor) {
    return processor(input);
}

function double(x) { return x * 2; }
function square(x) { return x * x; }
点击按钮查看结果

高级特性

3. 函数可以作为返回值

function createMultiplier(factor) {
    return function(x) {
        return x * factor;
    };
}

const triple = createMultiplier(3);
const quadruple = createMultiplier(4);
点击按钮查看结果

// 检查类型
