JavaScript 字符串连接运算符

+ 运算符注意事项

  • 用于字符串连接和加法运算
  • 操作数包含字符串时执行连接操作
  • 会进行自动类型转换
  • 操作顺序影响结果
  • 空字符串与数字相加会转换为字符串
  • 布尔值与字符串相加会转换为字符串

连接运算符示例

const str = "Hello";
const num = 123;

// 基本连接
console.log(str + " World"); // "Hello World"

// 类型转换
console.log(str + num);      // "Hello123"
console.log(num + str);      // "123Hello"
console.log(1 + 2 + str);    // "3Hello"
console.log(str + 1 + 2);    // "Hello12"

// 字符与数字相加
const char = "A";
console.log(char + 1);       // "A1"
console.log(1 + char);       // "1A"

// 空字符串与数字
console.log("" + 123);       // "123"
console.log(123 + "");       // "123"

// 布尔值与字符串
console.log(true + "1");     // "true1"
console.log("1" + false);    // "1false"

// 多操作数连接
const fullStr = str + " " + num + " " + true;
console.log(fullStr);        // "Hello 123 true"
