JavaScript String Quotes

注意事项

  • 字符串中可以使用单引号或双引号
  • 在单引号字符串中使用双引号不需要转义
  • 在双引号字符串中使用单引号不需要转义
  • 使用反斜杠(\)转义相同类型的引号
  • 模板字符串(``)可以包含任意引号

引号使用示例

// 单引号字符串包含双引号
const str1 = 'He said "Hello"';

// 双引号字符串包含单引号
const str2 = "It's a nice day";

// 转义相同类型的引号
const str3 = 'Don\\'t worry';
const str4 = "She said \"Hi\"";

// 模板字符串包含引号
const name = 'Alice';
const str5 = `"${name}" said 'Hello'`;
