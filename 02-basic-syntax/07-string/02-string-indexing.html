JavaScript String Indexing

注意事项

  • 字符串索引从0开始
  • 可以使用[]或charAt()访问特定位置的字符
  • length属性获取字符串长度
  • substring()和slice()可以提取子字符串
  • 字符串是不可变的，索引访问是只读的

基本索引操作

const str = "Hello JavaScript";

// 使用[]访问字符
console.log(str[0]);  // 'H'
console.log(str[6]);  // 'J'

// 使用charAt()
console.log(str.charAt(1));  // 'e'

// 获取字符串长度
console.log(str.length);  // 16

子字符串操作

// substring(start, end)
console.log(str.substring(6, 10));  // 'Java'

// slice(start, end) 
console.log(str.slice(-6));  // 'Script'

// 负索引
console.log(str.slice(6, -6));  // 'Java'
