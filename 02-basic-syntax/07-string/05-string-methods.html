JavaScript String Methods

注意事项

  • 字符串方法不会修改原字符串，而是返回新字符串
  • concat() 用于连接字符串
  • includes() 检查是否包含子串
  • indexOf() 返回子串位置
  • toUpperCase()/toLowerCase() 转换大小写
  • trim() 去除两端空白
  • replace() 替换子串
  • == 和 === 用于比较字符串
  • localeCompare() 用于本地化比较

字符串方法示例

const str1 = "Hello";
const str2 = "World";

// 连接字符串
const concatStr = str1.concat(" ", str2); // "Hello World"

// 检查包含
const hasHello = concatStr.includes("Hello"); // true

// 查找位置
const worldIndex = concatStr.indexOf("World"); // 6

// 大小写转换
const upperStr = str1.toUpperCase(); // "HELLO"
const lowerStr = str2.toLowerCase(); // "world"

// 去除空白
const spacedStr = "  trim me  ";
const trimmedStr = spacedStr.trim(); // "trim me"

// 替换
const replacedStr = concatStr.replace("World", "JavaScript"); // "Hello JavaScript"
