JavaScript String Basics

注意事项

  • 字符串可以用单引号(')或双引号(")定义
  • ES6引入了模板字符串(``)，支持多行字符串和变量插值
  • 字符串是不可变的，任何操作都会返回新字符串
  • 常用输出方法：console.log(), document.write(), window.alert()

字符串定义示例

// 单引号字符串
const str1 = 'Hello World';

// 双引号字符串 
const str2 = "Hello JavaScript";

// 模板字符串
const name = 'Alice';
const str3 = `Hello ${name}`;

字符串输出示例

// 控制台输出
console.log(str1); // Hello World

// 页面输出
document.write(str2); // Hello JavaScript

// 弹窗输出
window.alert(str3); // Hello Alice
