JavaScript 模板字符串

模板字符串特点

  • 使用反引号(`)定义
  • 支持变量嵌入: ${variable}
  • 支持表达式计算: ${1 + 1}
  • 自动保留换行和缩进
  • 比字符串拼接更简洁

基础示例

const name = "张三";
const age = 28;

// 变量嵌入
const greeting = `你好，${name}！`;

// 表达式计算
const info = `${name}今年${age}岁，明年就${age + 1}岁了`;

// 多行字符串
const profile = `
  姓名: ${name}
  年龄: ${age}
  状态: ${age >= 18 ? '成年' : '未成年'}
`;

console.log(greeting);
console.log(info);
console.log(profile);
