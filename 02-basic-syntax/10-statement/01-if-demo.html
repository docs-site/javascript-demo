JavaScript if语句

if语句注意事项

  • 基本语法
    • if (条件) { 代码块 }
    • if (条件) { 代码块 } else { 代码块 }
    • if (条件) { 代码块 } else if (条件) { 代码块 } else { 代码块 }
  • 条件转换规则
    • 0、""、null、undefined、NaN → false
    • 其他值 → true
  • 最佳实践
    • 使用严格比较(===)避免类型转换问题
    • 复杂条件使用括号明确优先级

if语句示例

// 1. 基本if语句
const age = 18;
if (age >= 18) {
    console.log("成年人");  // 会执行
}

// 2. if-else语句
const score = 59;
if (score >= 60) {
    console.log("及格");
} else {
    console.log("不及格");  // 会执行
}

// 3. 多条件判断
const temperature = 25;
if (temperature > 30) {
    console.log("炎热");
} else if (temperature > 20) {
    console.log("温暖");  // 会执行
} else if (temperature > 10) {
    console.log("凉爽");
} else {
    console.log("寒冷");
}

// 4. 条件转换示例
const name = "";
if (name) {
    console.log("名字存在");
} else {
    console.log("名字为空");  // 会执行
}
