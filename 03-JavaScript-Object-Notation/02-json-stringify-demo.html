JSON.stringify() 方法演示

JSON.stringify() 注意事项

  • 基本用法
    • 将JavaScript对象转换为JSON字符串
    • 支持对象、数组、字符串、数字、布尔值和null
  • 特殊处理
    • 函数、Symbol、undefined会被忽略
    • Date对象会被转换为ISO字符串
    • 循环引用会抛出错误
  • 参数说明
    • replacer: 过滤或转换属性
    • space: 控制缩进格式

JSON.stringify() 示例

// 1. 基本对象转换
const obj1 = { name: "张三", age: 25 };
const json1 = JSON.stringify(obj1);

// 2. 数组转换
const arr1 = [1, "two", true];
const json2 = JSON.stringify(arr1);

// 3. 特殊值处理
const obj2 = { 
    date: new Date(),
    func: function() {},
    undef: undefined
};
const json3 = JSON.stringify(obj2);

// 4. 使用replacer参数
const obj3 = { 
    name: "李四", 
    password: "secret", 
    age: 30 
};
const json4 = JSON.stringify(obj3, ["name", "age"]);

// 5. 格式化输出
const obj4 = { a: 1, b: { c: 2 } };
const json5 = JSON.stringify(obj4, null, 2);
