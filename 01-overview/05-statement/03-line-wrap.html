JavaScript 代码折行书写示例

JavaScript 是脚本语言，浏览器会在读取代码时，逐行地执行脚本代码。

正确的折行方式

let longString = "这是一个非常非常长的字符串" +
                 "被合理地分成多行书写";

let result = calculateSomething(
    param1,
    param2,
    param3
);

错误的折行方式

// 错误1: 在字符串中间直接换行
// let brokenString = "这个字符串
